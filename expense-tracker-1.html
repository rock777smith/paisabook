📱 SMS Auto-fill Setup

  1. Install MacroDroid from Play Store (free)
  2. Create a new Macro → Trigger: SMS Received
  3. Add condition: SMS contains "debited" OR "credited" OR "spent"
  4. Action: Open URL → Enter:
  5. In the URL, use MacroDroid variable {sms_body} to pass SMS content
  6. The app will auto-detect the amount and highlight the entry in yellow until you add a description

💡 Alternatively, use IFTTT or Automate — all free options.
